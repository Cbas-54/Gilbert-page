# Proyecto: Pagina Gilbert

## Que es
Landing page premium para servicios de Gilbert, con enfoque en estética moderna y experiencia de usuario fluida.

## Stack
- Frontend: React 19, Vite
- Estilo: Tailwind CSS v4 (vía `@tailwindcss/vite`)
- Iconos: Lucide React
- Navegación: React Router Dom v7

## Convenciones
- **Componentes**: Funcionales, en `src/components/`.
- **Estilos**: Preferir utilidades de Tailwind v4. Mantener consistencia visual "premium".
- **Archivos**: PascalCase para componentes, camelCase para hooks/utilidades.
- **Commits**: Conventional Commits (`feat:`, `fix:`, `refactor:`).

## Estructura
- `.antigravity/` - Mi configuración y skills.
- `docs/` - Arquitectura y decisiones (ADRs).
- `tools/` - Scripts y prompts.
- `src/components/` - UI Components.

## Reglas para Antigravity
- Mantener diseño "stunning" (Wow factor).
- No usar placeholders; generar imágenes reales.
- Documentar decisiones importantes en `docs/decisions/`.
- Cada módulo complejo debe tener su propio `instructions.md`.
