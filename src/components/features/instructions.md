# Modulo: Features

Componentes que implementan funcionalidades específicas de negocio o secciones complejas.

## Sub-módulos
- `home/`: Secciones de la landing principal (`Hero`).
- `products/`: Componentes relacionados con el catálogo (`ProductCarousel`).
- `location/`: Secciones de mapas y contacto físico.

## Reglas
- Cada feature puede tener su propia lógica de obtención de datos.
- Intentar desacoplar las features entre sí.
