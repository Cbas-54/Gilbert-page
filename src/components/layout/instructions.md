# Modulo: Layout

Componentes estructurales que definen el marco de la aplicación.

## Componentes
- `Navbar.jsx`: Navegación principal.
- `Footer.jsx`: Información de contacto y links legales.
- `Layout.jsx`: Contenedor principal con `Outlet`.

## Reglas
- Mantener el Navbar pegajoso (sticky) y con transiciones suaves.
- El Footer debe ser sobrio y contener toda la información de contacto.
